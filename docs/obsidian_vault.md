# Obsidian Vault

The generated vault lives in `atlas/vault`.

Entity pages contain YAML frontmatter and sections for:

- quick summary
- known state
- where to ask
- relations
- sources

Directory mapping:

- `clientes`: customers, contacts, quotes, invoices
- `productos`: products
- `proyectos`: projects and research projects
- `temas`: tickets, meetings, topics, risks, documents
- `decisiones`: decisions
- `procesos`: processes
- `ciclos`: cycles and proposals
- `proveedores`: suppliers

Vault files are generated artifacts. Update source exports and rebuild the vault instead of editing generated pages as canonical data.
